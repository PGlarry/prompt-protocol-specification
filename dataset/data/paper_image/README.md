# Paper Image Dataset — Structured Intent Encoding for AI Image Generation

**Paper:** "Structured Intent Encoding for AI Image Generation: Purpose Anchoring, Density Boundaries, and Cross-Modal Protocol Transfer"  
**Venue:** Neurocomputing (submitted 2026)  
**Authors:** Chaoyang Li, Gang Peng  
**Status:** Under review; data pre-released for reproducibility

---

## Overview

This dataset accompanies the cross-modal PPS/5W3H image-generation study. The paper investigates whether the 5W3H/PPS structured intent encoding framework — previously validated in text generation (Papers 1–4) — transfers to neural text-to-image (T2I) generation. The study uses three commercial Chinese T2I systems (Wanx, CogView, DoubaoImage) across 1,296 generated images in 324 experimental cells.

**Core finding:** PPS/5W3H exhibits cross-modal protocol-level effects in image generation, but its effectiveness is task-conditioned, component-selective, and density-bounded — not uniformly beneficial.

## Evaluation Metrics

| Metric | Description |
|--------|-------------|
| **fICMw** | Fidelity Intent Coverage weighted — whether the image preserves task-specific communicative intent per GIC dimensions |
| **sICMw** | Structural Intent Coverage weighted — whether a dimension is recognizably instantiated in the image |
| **Structural gap** | sICMw − fICMw; captures structurally plausible images that still fail intent fidelity |

Scoring: LLM-as-judge (DeepSeek-Chat, temperature=0) against pre-authored Gold Intent Cards (GICs). Each image scored twice (J1, J2); reported values are J1+J2 averaged per unique image.

**Dimension weights (pre-specified):** Why = 0.30, HowMuch = 0.20, Who = 0.15, HowFeel = 0.15, remaining = 0.20 shared.

## Conditions

### Base Conditions (Pilot + Exp A)

| ID | Name | Description |
|----|------|-------------|
| A | Free-form | Natural language; no dimensional scaffolding |
| C | Full PPS | Complete 8-dimension 5W3H, manually authored |
| D | AI-expanded PPS | PPS generated by DeepSeek-Chat from brief task description |

### Exp A — Ablation Conditions

| Condition | Meaning |
|-----------|---------|
| FULL | All 8 dimensions |
| ABL_WHY | Remove Why dimension |
| ABL_WHO | Remove Who dimension |
| ABL_HOWFEEL | Remove How-feel dimension |
| ABL_HOWMUCH | Remove How-much dimension |

### Exp C — Why Variants

| Condition | Why content | Char count |
|-----------|-------------|-----------|
| WHY_PURPOSE | Communicative goal only | ~18 chars |
| WHY_AUDIENCE | Recipient framing only | ~16 chars |
| WHY_BOTH | Purpose + audience | ~23 chars |
| WHY_ABSTRACT | Abstract value orientation | ~16 chars |
| NO_WHY | Why dimension omitted | — |

### Exp D — Density Conditions

| Condition | Active dims | Description |
|-----------|-------------|-------------|
| ELASTIC_3 | 3 | What + Why + one contextually relevant How |
| ELASTIC_5 | 5 | What + Why + Who + When/Where + one How |
| FULL_8 | 8 | All eight dimensions |
| STRICT_PLUS | 8+ | All eight + additional intra-dimensional constraints |

## Key Results Summary

### Pilot (Stage 0)
- T05 C vs A: Δ = +0.527 (95% CI [0.246, 0.816], d = 1.73) — strongest pilot effect
- T11 (low-ambiguity control): |Δ| < 0.005 across all conditions
- Structural gap persists across 18/18 pilot task-condition aggregates

### Why Decomposition (Exp C)
- T10 WHY_PURPOSE vs WHY_BOTH: Δ = +0.121 (95% CI [0.043, 0.203], d = 1.39) — CI excludes zero
- T08 WHY_AUDIENCE: 0.594 — below NO_WHY (0.695); audience specification is harmful in overload-sensitive tasks

### Density (Exp D)
- T08 STRICT_PLUS vs FULL_8: Δ = −0.453 (95% CI [−0.629, −0.270], d = −2.31) — strongest effect, 3/3 models consistent
- T08 Wanx STRICT_PLUS: fICMw = 0.094 — minimum across all 324 cells
- No single density condition is globally optimal (H-D4 confirmed): five tasks, four distinct optimal conditions

### Negative Result (Exp D-opt)
- No AI-optimized prompt-surface variant outperforms manual Condition C at image layer across any task
- Prompt-surface quality ≠ execution-layer intent fidelity

## Statistical Framework

- **Bootstrap CI:** 10,000 resamples, seed=42, on image-level J1+J2 averaged scores
- **n per comparison:** 8 averaged image-level observations (4 images/model × 2 primary models, after J1+J2 merge)
- **Effect size interpretation:** Cohen's d ≈ 0.2 small, 0.5 medium, 0.8 large (descriptive, not stand-alone confirmation)
- **Sign consistency:** Direction agreement across Wanx + CogView required for directional claims
